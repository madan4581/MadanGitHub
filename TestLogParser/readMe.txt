LogParser readMe

The given program parses the provided log file and generates output of URLS for the server "SJC"

1. Running the program
	a. Ensure Java is in the system path.
	b. Launching the program
	
	Syntax: java -jar logParser.jar <sourceLogFile>
	Example: java -jar logParser.jar  c:\urls.csv

	The source log file can also be placed in a folder called sourceLogFiles in the current directory.

	java -jar logParser.jar
